# 🐾 CasaPet - Sistema para PetShop

Este projeto consiste no desenvolvimento de uma aplicação para gerenciamento de um **PetShop**, criada utilizando uma plataforma **low-code (4Biz)**.

O sistema foi desenvolvido em parceria com as empresas **Run2Biz** e **CentralIT**, com foco em soluções práticas para o gerenciamento de serviços e clientes.

---

## 🚀 Projeto

O **CasaPet** é uma aplicação voltada para auxiliar no controle e organização de um PetShop, permitindo a gestão de informações de forma eficiente.

---

## 🛠️ Tecnologias e Ferramentas

- ⚙️ Plataforma Low-Code (4Biz)  
- 🗂️ Estrutura baseada em aplicações corporativas  
- 📄 Arquivo de exportação do sistema (`.json`)
